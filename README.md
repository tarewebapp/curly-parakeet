# T-ARE Trading and Consultancy, Inc.

Official corporate website for **T-ARE Trading and Consultancy, Inc.**, a Philippine-based company specializing in engineering and technology solutions.

## Overview

This is a modern, high-performance, mobile-first static website built with vanilla HTML, CSS, and JavaScript. The site showcases T-ARE’s expertise in solar energy, fire detection systems (FDAS), EV charging infrastructure, enterprise networking, CCTV, POS systems, and custom software development.

## Key Features

- **Cinematic Hero Section** — Features advanced LQIP (Low Quality Image Placeholder), blur-up effect, subtle scale animation, vignette, and grain texture overlay
- **Advanced Logo Interactions** — 3D tilt effect on mouse movement + premium hover animations
- **Dark Mode** — Full theme support with system preference detection and localStorage persistence
- **Accessibility Optimized** — Proper form labels, autocomplete attributes, and semantic markup
- **Print-Ready Styles** — Clean, professional print stylesheet
- **Complete Brand Assets** — Includes primary logo, horizontal logo, favicon set, LinkedIn banner, business card mockup, and email signature assets
- **Brand Guidelines** — Professional one-page brand guidelines (`brand-guidelines.html`)

## Tech Stack

- Pure vanilla HTML5, CSS3, and ES6 JavaScript (no frameworks)
- Responsive design with mobile-first approach
- CSS custom properties for theming
- Optimized images (PNG + WebP)

## Getting Started

Simply open `index.html` in any modern browser. No build step or server required.

For local development, you can use any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```
